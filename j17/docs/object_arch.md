# 实例内存布局

## object header 8 byte

```plain
|---------------------------------------------------------------|---------|
| Object Header (64 bits)                                       | State   |
|---------------------------------------------------------------|---------|
| Mark Word(32 bits)                     | Class Word (32 bits) |         |
|---------------------------------------------------------------|---------|
|                               | lock:2 | class meta           | Normal  |
|---------------------------------------------------------------|---------|
| monitor ptr:30                | lock:2 | class meta           | Locked  |
|---------------------------------------------------------------|---------|
| gc meta:30                    | lock:2 | class meta           | GC      |
|---------------------------------------------------------------|---------|
```

## instance data

```plain
long, double 64 bit 
...
int, char, reference, float 32 bit
...
short 16 bit
...
byte, 8 bit
...
boolean 1 bit
```

## padding
